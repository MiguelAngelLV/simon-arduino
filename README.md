Simon Arduino
=============

Simon para Arduino de cuatro colores.

Recuerda que el zumbador debe ir conectado a un pin PWM, si tu modelo de arduino no lo tiene en el PIN 9 selecciona el pin correspondiente la llamada al tone.

El esquema lo tenéis en png y en Fritzing.

Si hay algun problema con los componentes del fichero de la PCB para EAGLE, usa la libreria de Adafruit que se puede descargar en su Github

Se ha cambiado la distribucion de pines original para que la placa sea facilmente routeable a 1 cara.

Si solo quieres realizar la PCB pero no quieres editarla, puedes usar el fichero PDF que contiene el trazado de las pistas y pasarlo a tu placa de la manera que mas te guste.
